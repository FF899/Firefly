# 1.系统快捷键
Ctrl + Alt + T 打开一个新的命令行终端。  
Alt + F4 关闭当前窗口。  
Alt + F11 最大化当前窗口。  
Alt + F3 打开应用 程序 查找器。  
PrtSc 使用此快捷方式截取屏幕截图。  
Super + PrtSc 我们使用这个快捷方式打开 Kazam（截图/截屏工具）  
Ctrl + N 新的窗口  
Ctrl + X 切换窗口  
Ctrl + C 复制。  
Ctrl + Z 撤消。  
Ctrl + S 式用于保存  
Ctrl + Q 退出。  
F1 我们使用此快捷方式获得在线帮助。  
F10 左键单击菜单或光标  
Alt + F1 使用这个快捷方式，我们可以打开窗口。  
Alt + F2 使用这个快捷方式，我们可以打开命令窗口。  
Alt + F7 使用这个快捷方式，我们可以移动窗口。  
Alt + 空格 使用这个快捷方式，我们可以激活窗口菜单。  
Alt + F10 切换最大化状态。  
Ctrl + Alt + D 使用此快捷方式，我们可以最小化所有窗口。  
Alt + F8 借助此快捷方式，我们可以调整窗口大小。

# 2. 终端快捷键 
Ctrl + D 使用这个快捷方式，我们可以关闭终端窗口。  
Ctrl + + 此快捷方式用于放大。  
Ctrl ± 此快捷方式用于缩小。  
Ctrl + L 清除终端  
Ctrl + C 使用这个快捷方式，我们可以取消当前正在运行的命令。  
Ctrl + A 使用这个快捷方式，我们可以将光标移动到行首。  
Ctrl + B 使用此快捷方式，我们可以将光标向后移动到一个字符。  
Ctrl + E 使用这个快捷方式，我们可以将光标移动到行尾。  
Ctrl + F 使用此快捷方式，我们可以将光标向前移动一个字符。

---
# 3. 截图快捷键
Print Screen 使用这个快捷按钮，我们可以截图。  
Ctrl + Print Screen 截图到剪贴板  
Alt + Print Screen 使用此快捷方式，我们可以截取窗口的屏幕截图。  
Ctrl + Shift + Print Screen 借助此快捷方式，我们可以将某个区域的屏幕截图复制到剪贴板。  
Ctrl + Alt + Print Screen 借助此快捷方式，我们可以将窗口的屏幕截图复制到剪贴板。  
Shift + Ctrl + Alt + R 使用此快捷方式，我们可以录制截屏视频。

---
# 4. 工作区快捷键 
Shift + Ctrl + Alt + ↑ 我们可以将窗口向上移动一个工作区。  
Shift + Ctrl + Alt + 向下翻页 我们可以将窗口向下移动一个工作区。  
Shift + Ctrl + Alt + ← 我们可以将窗口向左移动一个工作区。  
Shift + Ctrl + Alt + → 我们可以将窗口向右移动一个工作区。
