---

**分类依据：** [WHATWG HTML Living Standard](https://html.spec.whatwg.org/multipage/#toc-semantics)

**替代方案：** 
- `<font>` → CSS `font-family` 
- `<center>` → CSS `text-align` / flexbox 
- `<strike>` → `<s>` 或 `<del>`
- `<marquee>` → CSS animation
- `<big>` → CSS `font-size`
- `<tt>` → `<code>` 或 CSS `font-family: monospace`
